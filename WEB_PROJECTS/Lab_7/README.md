# PHP Beginner Lab Tasks (Lab 7)

Welcome to the beginner PHP tasks repository! This project contains a collection of 10 fundamental PHP scripts designed to introduce and reinforce core PHP programming concepts. It's a great starting point for someone learning PHP, covering everything from basic variables and data types to arrays, loops, and custom functions.

## 📝 Tasks Overview

Here is a breakdown of what each task file contains and the concepts it demonstrates:

- **`task_01.php` - Variables and Constants:** Demonstrates how to declare integers, floats, strings, and constants (`define()`), along with string concatenation and basic output using `echo`.
- **`task_02.php` - String Manipulation:** Utilizes built-in PHP string functions such as `strrev()` (reversing a string), `str_word_count()` (counting words), and `stripos()` (finding the position of a substring).
- **`task_03.php` - Functions and Exception Handling:** Shows how to define custom functions with parameters, validate parameter types (`is_int()`), and throw exceptions (`InvalidArgumentException`).
- **`task_04.php` - Nested Loops:** Uses nested `for` loops to generate and print a "half-diamond" number pattern.
- **`task_05.php` - PHP and HTML Integration:** Demonstrates how to output HTML structures dynamically using PHP, specifically rendering a formatted HTML table with PHP variable values.
- **`task_06.php` - Array Filtering:** Shows how to work with indexed arrays and remove duplicate values using the built-in `array_unique()` function.
- **`task_07.php` - Advanced Loops & Math:** Implements a custom function to check for prime numbers, iterating through numbers up to 100 to print the primes and calculate their total sum.
- **`task_08.php` - Associative Arrays:** Introduces associative arrays (key-value pairs) and demonstrates how to iterate through them using a `foreach` loop.
- **`task_09.php` - Mathematical Logic:** Contains a custom function that uses `while` loops and modulo operators (`%`) to calculate the sum of the individual digits of a given number.
- **`task_10.php` - Array Operations & Sorting:** Covers advanced array manipulation, including converting arrays to strings (`implode()`), sorting alphabetically (`sort()`), adding new elements (`array_push()`), and rendering the array elements as HTML unordered lists (`<ul>`).

## 🚀 How to Run the Code

To execute these PHP scripts, you will need a local PHP development environment.

1. **Install a Local Server:** Download and install a local server package like [XAMPP](https://www.apachefriends.org/index.html), [WAMP](https://www.wampserver.com/en/), or [MAMP](https://www.mamp.info/).
2. **Start the Server:** Launch the application and start the **Apache** service.
3. **Move the Project:** Copy this entire `LAB 7` folder into your local server's document root directory:
   - For **XAMPP**: `C:\xampp\htdocs\`
   - For **WAMP**: `C:\wamp\www\`
4. **View in Browser:** Open your web browser and navigate to `http://localhost/LAB 7/task_01.php` (replace `task_01.php` with whichever file you want to test).
